# AI & Data Science Practicals

## 1. AI Problem Identification & Solution Design
Identify a real-world industry problem and propose an AI-driven solution, including methodology, expected impact, and implementation feasibility.

## 2. AI Model Implementation Using TensorFlow/PyTorch
Build and train a basic AI model using popular frameworks, evaluate its performance using suitable datasets and metrics.

## 3. Sentiment Analysis Model
Develop a sentiment analysis system to analyze customer reviews and classify them as positive, negative, or neutral.

## 4. AI Deployment Strategy Comparison
Compare Cloud AI and Edge AI deployment strategies based on cost, performance, latency, scalability, and security.

## 5. Business Model & Ethical Analysis of AI Products
Design a Business Model Canvas for an AI product and analyze ethical challenges with proposed mitigation strategies.

## 6. Building and Testing an AI Application
Develop and evaluate an AI model for a practical application such as recommendation systems or spam detection.

## 7. AI System Architecture Design
Design the architecture of an AI-based product integrating cloud and edge computing components.

## 8. AI Security Implementation
Implement security measures to protect AI systems from adversarial attacks and data breaches.

## 9. Social Media Sentiment Analysis for Financial Markets
Analyze social media sentiment trends using AI to gain insights into financial market behavior.

## 10. AI Deployment Environment Comparison
Study and compare AI model deployment across cloud, on-premise, and edge environments focusing on latency, scalability, and cost.
